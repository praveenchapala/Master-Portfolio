# 📌 About The Project

Master Portfolio is a personal developer portfolio built from scratch using modern frontend technologies.

The goal of this project is to provide recruiters, hiring managers, and technical interviewers with a structured overview of my technical skills, projects, certifications, learning journey, and professional experience.

Instead of being just another portfolio website, this project is designed to reflect my engineering mindset, attention to detail, and passion for building real-world software solutions.

---

# 🎯 Objectives

This portfolio aims to:

- Showcase real-world software engineering projects
- Present technical skills in a structured manner
- Highlight my learning journey
- Demonstrate frontend development skills
- Provide quick access to GitHub repositories
- Make resume downloading easy
- Offer recruiters a professional overview within 30 seconds
- Help technical interviewers explore my work in depth

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/master-portfolio.git
```

Go to the project

```bash
cd master-portfolio
```

Navigate to the application

```bash
cd portfolio-app
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```

---

# 📦 Production Build

Generate production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
