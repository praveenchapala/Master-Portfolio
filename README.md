# 👨‍💻 About

Master Portfolio is my personal developer portfolio designed from scratch using **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

Rather than being just another portfolio website, this project demonstrates my ability to:

- Design modern user interfaces
- Build reusable React components
- Organize scalable project architecture
- Document software professionally
- Deploy production-ready applications

The portfolio showcases my projects, technical skills, certifications, internships, and software engineering journey.

# 🚀 Getting Started

Clone Repository

```bash
git clone https://github.com/praveenchapala/Master-Portfolio.git
```

Go inside project

```bash
cd Master-Portfolio/portfolio-app
```

Install packages

```bash
npm install
```

Run locally

```bash
npm run dev
```

Create Production Build

```bash
npm run build
```

# 💡 Challenges Faced

Developing this portfolio involved solving several real-world problems:

- Designing a modern UI from scratch.
- Structuring reusable React components.
- Maintaining a scalable folder architecture.
- Creating responsive layouts across devices.
- Implementing animations using Framer Motion.
- Fixing Hero section alignment issues.
- Debugging duplicate component rendering.
- Resolving Git authentication and permission issues.
- Configuring Vercel deployment for a nested project directory.
- Writing professional project documentation.

Each challenge improved my debugging, frontend architecture, and deployment skills.
